Web Refinery

Web refinery is an app that allows users to create a webpage through a simple drag and drop interface. Users can edit the content or add custom images to create a unique page. Once finished, users can view or export their project which includes html, css, images, and other assets (sass, gulp, package info) that more technical users will appreciate.

Role

Technology

  • Node
  • Express
  • Angular
  • jQuery
  • PostgreSQL
  • HTML/CSS
